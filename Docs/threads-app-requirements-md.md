# Threads自動投稿アプリ要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
Threads投稿マネージャー

### 1.2 概要
Threadsへの投稿を自動化するGoogle Apps Script（GAS）ウェブアプリケーション。複数アカウント管理と投稿スケジューリング機能を持ち、モバイルデバイスからのアクセスに最適化する。Google Driveと連携してメディアファイルを取得・投稿する。

## 2. 目的

- Threadsプラットフォームへの投稿を自動化
- 定期的なコンテンツ配信の効率化
- 手動投稿の手間とリソースの削減
- 複数アカウントの一元管理と効率的な運用
- モバイルデバイスからの操作を可能にし、場所を選ばず投稿管理を実現
- Google Driveのメディアファイルを活用した効率的なコンテンツ管理

## 3. 主要機能

### 3.1 投稿管理
- 指定した時間/間隔での自動投稿
- テキスト、画像、動画などのコンテンツタイプのサポート
- 投稿スケジュールの管理と編集
- 投稿成功/失敗の通知システム

### 3.2 複数アカウント管理
- アカウント情報の安全な保存と管理
- アカウントごとの投稿設定カスタマイズ
- 同一コンテンツの複数アカウントへの一括投稿機能
- アカウントごとのパフォーマンス追跡

### 3.3 Google Drive連携
- Google Driveからの画像・動画ファイルの参照と選択
- フォルダ単位でのメディアファイル管理
- ファイルプレビュー機能
- メディアファイルのキャッシュと最適化

### 3.4 モバイル対応
- スマートフォンに最適化されたレスポンシブUI
- タッチ操作のしやすさを重視したインターフェース
- 重要な機能へのアクセスしやすさ
- プッシュ通知または通知メール機能

## 4. 技術要件

### 4.1 開発プラットフォーム
- Next.js 15（React 19ベース）
- TypeScript
- Tailwind CSS + shadcn/ui コンポーネントライブラリ
- Vercelまたは類似プラットフォームでのデプロイ

### 4.2 データ管理
- 現在はReactのuseStateによるフロントエンド状態管理
- モックデータによる開発・テスト
- 将来的にはAPI連携またはデータベース統合予定

### 4.3 API連携
- Threads APIとの連携（非公式APIまたは公式APIが利用可能な場合）
- Google Drive APIとの連携
- OAuth認証または適切な認証方式の実装

### 4.4 モバイル最適化
- レスポンシブデザインの実装
- タッチフレンドリーなUI要素
- 画面サイズに応じた要素の再配置
- 低速ネットワーク環境を考慮した軽量設計

## 5. ファイル構成

### 5.1 Next.js アプリケーション構造
- `threads-app/` - メインアプリケーションディレクトリ
- `app/layout.tsx` - Next.js App Routerのルートレイアウト
- `app/page.tsx` - メインページ（ThreadsManagerコンポーネントを表示）
- `threads-manager.tsx` - 全ダッシュボード機能を含む統合コンポーネント

### 5.2 UIコンポーネント
- `components/ui/` - shadcn/uiベースのUIコンポーネント
  - `avatar.tsx` - アバター表示
  - `badge.tsx` - ステータス表示
  - `button.tsx` - ボタンコンポーネント
  - `card.tsx` - カード表示
  - `checkbox.tsx` - チェックボックス
  - `dialog.tsx` - モーダルダイアログ
  - `input.tsx` - 入力フィールド
  - `label.tsx` - ラベル
  - `radio-group.tsx` - ラジオボタングループ
  - `textarea.tsx` - テキストエリア

### 5.3 設定ファイル
- `package.json` - 依存関係とスクリプト定義
- `tsconfig.json` - TypeScript設定
- `eslint.config.mjs` - ESLint設定
- `postcss.config.mjs` - PostCSS設定
- `next.config.ts` - Next.js設定
- `components.json` - shadcn/ui設定

### 5.4 スタイル
- `app/globals.css` - グローバルCSSとTailwind CSS設定
- `lib/utils.ts` - Tailwind CSS ユーティリティ関数
